# TaskReport Dart

## Sobre o projeto

O TaskReport Dart é um simulador de tratamento de dados de tarefas em Dart.

O miniprojeto simula o retorno de uma API por meio de uma lista de mapas, tratando os dados e os transformando em objetos, e gerando relatórios com base neles.

## Objetivo 

Praticar os principais conceitos da parte de linguagem Dart do Módulo 01 do curso de Desenvolvimento Mobile.

## Estrutura do projeto
```
taskreport-dart/
├── planejamento/
│   ├── tarefas-kanban.md
├── dados_tarefas.dart
├── item.dart
├── main.dart
├── README.md
├── tarefa.dart
```

## Como Executar

Tendo o dart SDK instalado e configurado, utilize o terminal para navegar até a pasta do projeto 'taskreport-dart' baixada ou clonada e execute o arquivo 'main.dart' através do terminal com o comando 
```
dart run main.dart
```
Seram exibidos no terminal os relatórios pedidos na documentação do miniprojeto.
