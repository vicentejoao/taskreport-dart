# TaskReport Dart

## Sobre o projeto

O TaskReport Dart é um simulador de tratamento de dados de tarefas em Dart.

O projeto simular o retorno de uma API por meio de uma lista de mapas, transforma esses dados em objetos e gera relatórios com base nos status das tarefas.

## Objetivo 

Praticar os principais conceitos da parte de linguagem Dart do Módulo 01 do curso de Desenvolvimento Mobile.

## Como Executar

Navegue até a pasta do projeto 'taskreport-dart' e execute o arquivo 'main.dart' através do terminal com o comando 
```
dart run main.dart
```
